Данная задача - это продолжение миссии Cheapest Flights

Ваша задача, имея расписание самолетов узнать, все ли рейсы действительно необходимы.

Например: Если из точки А в точку С можно долететь напрямую на $100, а можно и через точку B (из А в B и из B в C) за $90, то соединение A-C - не нужно. Но если цена из А в C будет $90 или меньше, то это соединение остается в расписании.

Input: Массив всех возможных соединений. Каждый элемент в массиве - это тоже массив из 3х элементов: первый город, второй город и цена перелета.

Output: Массив индексов перелетов, которые являются ненужными.

Example:

  ['B', 'C', 40],
  ['A', 'C', 100]]) == [2]
uselessFlight([['A', 'B', 50],
  ['B', 'C', 40],
  ['A', 'C', 90]]) == []
